# Todo App Frontend

This is the React frontend for the Todo App. It connects to the backend API and provides a modern Trello-like task board.

## Features
- Add, update, delete tasks
- Move tasks between categories (To Do, In Progress, Done)
- Responsive design
- Deployed on Vercel

## Tech Stack
- React
- Axios
- Vercel (Deployment)
